# Job Card Manager (React)

A responsive Job Card Manager built using React and Tailwind CSS.
Users can create, edit, delete, and persist job cards using localStorage.

## Features
- Add job cards
- Edit existing cards
- Delete cards
- Mobile modal form
- Desktop side form
- localStorage persistence
- Responsive grid layout

## Tech Stack
- React
- Tailwind CSS
- Vite
- localStorage

## Project Structure
App.jsx handles global state.
Form component is reusable for create and edit.
CardsSection renders job cards grid.
JobCard displays individual card.
FormModal is used only on mobile devices.

## Learning Outcome
- State lifting
- Controlled forms
- Conditional rendering
- Responsive UI logic
- localStorage handling
